本节心得
感觉自己还不能够很好的梳理出来自己的逻辑。只是可以从里面抽出来一些比较有用的知识点。用我认同的查理芒格的话，就是说，如果我知道我会死在哪里，那我怎么都不会去那里——我会绕着走。
查理芒格所谓的，用各个学科的核心知识框架打造的思维格栅理论，我想也就是为了增强人类的理性，从而能够做出更加明智的决策。
因此，我们学习查理芒格的思维格栅，不如以斯坦诺维奇现在创建的这个高效的框架为基础，在这里面添加一些我们通过读书学到的新东西，丰富到这些表格里来。也就是可以做一个“不理性清单”（避免自己愚蠢的清单）。我们明白这些原理，然后此后的人生大框架的重要决定要检查一下这个不理性清单，将自己的逐渐变得更聪明起来。理性是有复利效应的，我想，我们如果有志于从事非虚构性写作，通过写作的过程让自己变得更“理性”，那么这将是一个非常棒的脚手架。

# 认知偏见列表

维基百科，自由的百科全书

[跳转到导航](https://en.wikipedia.org/wiki/List_of_cognitive_biases#mw-head)[跳转到搜索](https://en.wikipedia.org/wiki/List_of_cognitive_biases#searchInput)

[认知偏差](https://en.wikipedia.org/wiki/Cognitive_bias)是偏离规范和/或理性判断的系统模式。他们经常在[心理学](https://en.wikipedia.org/wiki/Psychology)和[行为经济学中学习](https://en.wikipedia.org/wiki/Behavioral_economics)。[[1\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-1)

尽管这些偏见中的大多数已通过可[重复的](https://en.wikipedia.org/wiki/Reproducible)研究得到了证实，[[2\] ](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-2)[[3\]仍](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-3)存在关于如何对这些偏见进行分类或如何解释的争议。[[4\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-4)几个[理论原因是已知的，一些认知偏差](https://en.wikipedia.org/wiki/Cognitive_bias#Common_theoretical_causes_of_some_cognitive_biases)，其通过它们的共同生成机制（如嘈杂信息处理提供偏压的分类[[5\] ](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-HilbertPsychBul-5)）。[格尔德·吉格伦泽（Gerd Gigerenzer](https://en.wikipedia.org/wiki/Gerd_Gigerenzer)）批评将认知偏差的框架[归结](https://en.wikipedia.org/wiki/Gerd_Gigerenzer)为判断错误，并主张将其解释为源自逻辑思维的理性偏离。[[6\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-6)

解释包括大脑用来产生[决策](https://en.wikipedia.org/wiki/Decision-making)或判断的信息处理规则（即思维捷径），称为*[启发法](https://en.wikipedia.org/wiki/Heuristics)*。偏见具有多种形式，并表现为认知（“冷”）偏见，例如精神障碍[[5\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-HilbertPsychBul-5)或动机（“热”）偏见，例如，信念因[一厢情愿](https://en.wikipedia.org/wiki/Wishful_thinking)而扭曲时。两种效果可以同时出现。[[7\] ](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-7)[[8\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-8)

关于这些偏见是否存在无用还是[不合理](https://en.wikipedia.org/wiki/Irrationality)，还是导致有用的态度或行为，也存在争议。例如，在结识他人时，人们往往会提出[引导性问题](https://en.wikipedia.org/wiki/Leading_question)，这些[问题](https://en.wikipedia.org/wiki/Leading_question)似乎倾向于证实他们对这个人的假设。但是，这种[确认偏差](https://en.wikipedia.org/wiki/Confirmation_bias)也被认为是[社交技巧的](https://en.wikipedia.org/wiki/Social_skills)一个例子。与他人建立联系的一种方式。[[9\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-dardenne-9)

尽管这项研究绝大多数涉及人类受试者，但在非人类动物中也发现了一些证明存在偏见的发现。例如，已经在猴子中显示出[厌恶损失](https://en.wikipedia.org/wiki/Loss_aversion)，并且在大鼠，鸽子和猴子中观察到了[双曲线折扣](https://en.wikipedia.org/wiki/Hyperbolic_discounting)。[[10\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-10)

| [第三人称效果](https://en.wikipedia.org/wiki/Third-person_effect) | [自我中心偏见](https://en.wikipedia.org/wiki/Egocentric_bias) | 一种假设的趋势是，相信大众传播的媒体消息对他人的影响大于对自己的影响。截至2020年，第三人称效果尚未在科学背景下得到可靠证明。 |
| [最终归因错误](https://en.wikipedia.org/wiki/Ultimate_attribution_error) |  [归因偏差](https://en.wikipedia.org/wiki/Attribution_bias)  | 类似于基本归因错误，在此错误中，一个人可能对整个组而不是组中的个人进行内部归因。 |
| [差于平均的效果](https://en.wikipedia.org/wiki/Worse-than-average_effect) |                                                              | 倾向于相信自己在困难的工作上会比其他人差。[[119\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-Kruger,_J._1999-119) |

## 记忆[[编辑](https://en.wikipedia.org/w/index.php?title=List_of_cognitive_biases&action=edit&section=3)]

主条目：[内存偏差列表](https://en.wikipedia.org/wiki/List_of_memory_biases)

在[心理学](https://en.wikipedia.org/wiki/Psychology) *和* [认知科学中](https://en.wikipedia.org/wiki/Cognitive_science)，记忆偏见是一种[认知偏见](https://en.wikipedia.org/wiki/Cognitive_bias)，它会增强或削弱记忆的[回忆](https://en.wikipedia.org/wiki/Memory)（完全消除记忆的机会，或记忆被回忆所花费的时间，或者两者），或更改已报告内存的内容。有多种类型的内存偏差，包括：

| :----------------------------------------------------------: | :----------------------------------------------------------: |

|                          一致性偏差                          | 错误地记住一个人过去的态度和行为，就像现在的态度和行为。[[123\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-123) |
| [上下文效应](https://en.wikipedia.org/wiki/Cue-dependent_forgetting) | 认知和记忆取决于上下文，因此，与上下文内存相比，上下文外内存更难检索（例如，与工作相关的内存的召回时间和准确性在家里会更低，反之亦然） 。 |

| [褪色影响偏见](https://en.wikipedia.org/wiki/Fading_affect_bias) | 与不愉快的记忆有关的情绪比与积极事件有关的情绪更容易消失的偏见。[[125\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-125) |
|   [错误的记忆](https://en.wikipedia.org/wiki/False_memory)   |        一种*错误归因*形式，其中想象力被误认为是记忆。        |
| [世代效应](https://en.wikipedia.org/wiki/Generation_effect)（自世效应） | 最好记住自己生成的信息。例如，与其他人产生的类似陈述相比，人们更容易回忆起他们所产生的陈述的记忆。 |
|   [谷歌效应](https://en.wikipedia.org/wiki/Google_effect)    |   倾向于忘记可以使用Internet搜索引擎在网上轻松找到的信息。   |
|                           幽默效果                           | 与非幽默的项目相比，幽默项目更容易被记住，这可以用幽默的独特性，理解幽默的认知处理时间的增加或幽默引起的情感唤起来解释。[[126\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-126) |
|                           滞后效应                           | 与在单个会话中学习相同时间量不同，学习时学习量更大的现象会随着时间的流逝而扩展。另请参见[间距效果](https://en.wikipedia.org/wiki/Spacing_effect)。 |
| [校平和锐化](https://en.wikipedia.org/wiki/Leveling_and_sharpening) | 记忆失真是由于随着时间的流逝而丢失细节而引起的，通常与锐化或选择性地恢复某些细节有关，这些细节相对于因练级而丢失的体验的细节或方面具有夸大的意义。随着时间的流逝，以及通过反复记忆或记忆，这两种偏见都可能得到加强。[[127\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-127) |

|                         列表长度效应                         | 在较长的列表中记住的项目百分比较小，但是随着列表长度的增加，记住的绝对项目数也会增加。例如，考虑30个项目的列表（“ L30”）和100个项目的列表（“ L100”）。个人可能会记住L30的15个项目，即50％，而个人可能会记住L100的40个项目，即40％。尽管记忆的L30项百分比（50％）大于记忆L100的百分比（40％），但记忆的L100项（40）比记忆L30的项（15）多。[[129\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-memcog-129) [*[需要进一步说明](https://en.wikipedia.org/wiki/Wikipedia:Please_clarify)*] |
| [错误信息效应](https://en.wikipedia.org/wiki/Misinformation_effect) | 由于*事件后信息*的干扰，内存的准确性下降。[[130\]](https://en.wikipedia.org/wiki/List_of_cognitive_biases#cite_note-130) |
|  [情态效应](https://en.wikipedia.org/wiki/Modality_effect)   | 当通过语音接收列表项时，与通过书写接收列表项时相比，列表最后一项的记忆调用率更高。 |
